Bobby's World
- Unity Game

For this projected I developed a small game in Unity called Bobby's World, using already existing 3D assets, creating a enjoyable environment with interactions, using C# as the coding languge. It served as a way to put to work my coding knowledge and learn how to use the basis of Unity.

01 Overview

•

Bobby's World

In this game the camera follows the character around the environment maintaining the same distance at all times. When the character walks or runs there are footstep sounds aligned with each footstep, as well as a jumping animation when he jumps. For the environment and it's intercations, there is a background music playing while Bobby is exploring. There are also several coins which he can catch and make a sound as they are being collected, as well as a coin count in the console. Besides, there's a tresoure box that he can open.
